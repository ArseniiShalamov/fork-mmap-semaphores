# Fork, Shared Memory and Semaphores

This repository contains several Operating Systems assignments implemented in C.

## Contents

### myshell.c
A simple Unix shell implementation using:
- fork()
- exec()
- wait()

### H.W.2.c
Process synchronization assignment using System V semaphores.

### H.W.2-3.c
Multi-process synchronization and ordered execution using semaphores.

### H.W.3.c
Shared memory and semaphore synchronization between processes.

## Topics

- Process creation with fork()
- Process synchronization
- Semaphores
- Shared memory
- Inter-Process Communication (IPC)
- Shell implementation

## Environment

Developed and tested on Unix/Linux systems using the C programming language.
